This repository is the official implementation containing the source code and datasets for the paper:

**"Structure-Adaptive Conformal Recommendation: Knowledge Graph Topology as an Evidential Prior for Tighter Prediction Sets"** 
*(Currently under consideration at Knowledge-Based Systems)*.

## Important Notice: Code & Data Availability

Thank you for your interest in our work!

To comply with the strict **double-blind review process** and the journal's submission policies, the full source code, preprocessed datasets (Amazon-Book, Last-FM, MovieLens-1M), and pre-trained model checkpoints are temporarily withheld from this repository.

**All materials (Code + Data) will be made fully public and open-source immediately upon the paper's acceptance.**

In the meantime, you can find the detailed theoretical proofs, algorithm design, and hyperparameter configurations within the manuscript.

### What to expect upon release:
- Complete PyTorch implementation of the **`KGUP`** evidential uncertainty probe.
- The **`SA-OKAC`** online multi-scale feedback calibration algorithm.
- Scripts for Knowledge Graph topology pre-processing (Jaccard overlaps, Fiedler algebraic connectivity).
- End-to-end evaluation pipeline replicating the results in the paper.
- Preprocessed datasets and PyTorch geometric data loaders.
